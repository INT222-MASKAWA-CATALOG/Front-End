# MASKAWA
```
Front-End หลัก ๆ จะใช้ Vue Router ในการทำ Single Page Application (SPA) 
และใช้ Tailwind.css ในการช่วยเสริมเรื่องของการจัด layout ต่าง ๆ 
รวมไปถึงการตกแต่งหน้า Web Application ให้น่าสนใจมากขึ้น
```
## Project setup
```
npm install
```
